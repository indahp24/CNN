# 🧠 Brain Tumor Classification using CNN

## 📌 Overview

This project aims to classify brain MRI images into different tumor categories using **Convolutional Neural Networks (CNN)**. The model learns visual patterns from medical images to automatically distinguish between tumor types.

---

## 🗂️ Dataset

The dataset consists of brain MRI images categorized into:

* Glioma
* Meningioma
* Tumor

The dataset is split into:

* Training set
* Validation set
* Testing set

---

## 🏗️ Model Architecture

The following models are implemented and evaluated:

* MobileNetV2

Training includes:

* Image preprocessing
* Data augmentation
* Model optimization using TensorFlow/Keras

---

## ⚙️ Tools & Technologies

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* Scikit-learn
* Google Colab

---

## 📊 Results

The model achieved strong performance on the test dataset with high accuracy.

Example results:

* Accuracy: ~97% – 99%
* Confusion Matrix indicates minimal misclassification

---

## 📈 Evaluation Metrics

Model performance is evaluated using:

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix
